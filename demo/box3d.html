When you display many content that all belongs together, you normally "group" this content visually. Box3D is the way to do this.

All elements can be turned into a box widh a 3d outline via .gh-box3d css-class.



And most of the time such a widget will have a title (.gh-title, .gh-title-video, .gh-title-audio) and a media player. Although you can put any other html code inside a widget like this, it is not very common.

This is a .gh-title-video

The only other media-type, that you can use, is .gh-title-audio

As you can see, the only difference is the icon in front of the title description.

But these .gh-box3d elements can also contain other arbitrary content, including all kinds of html markup. There's really no limit! - Although, you should try to fallow the main visual guides of Geekhunger as close as you can! This is better for the user experience.

Box3D's can also be nested! And colored!
